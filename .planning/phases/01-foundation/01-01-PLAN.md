<task type="auto">
  <name>Task 2: Implement Frame, FrameError, DisplayEnvironment with unit tests</name>
  <files>src/protocol.rs</files>
  <action>
Implement the wire protocol types in `src/protocol.rs`. This is the critical data layer — all frame serialization logic lives here, and it must be synchronous (no async, no I/O).

1. **Constants:**
   - `pub const MAGIC: [u8; 2] = [0xC5, 0x53];` — 0xC5 is non-ASCII (prevents confusion with text), 0x53 is 'S'
   - `pub const VERSION: u8 = 1;`
   - `pub const FRAME_TYPE_PNG: u8 = 0x01;`
   - `const HEADER_LEN: usize = 8;` — 2 magic + 1 version + 1 type + 4 length

2. **FrameError enum** with `#[derive(Debug, thiserror::Error)]`:
   - `#[error("invalid magic bytes")] InvalidMagic`
   - `#[error("unsupported protocol version: {0}")] UnsupportedVersion(u8)`
   - `#[error("frame data too short")] TooShort`
   - `#[error("frame payload exceeds maximum size ({0} bytes)")] TooLarge(usize)`
   - `#[error("payload length mismatch: header says {expected} bytes, got {actual}")] LengthMismatch { expected: usize, actual: usize }`

3. **DisplayEnvironment enum** with `#[derive(Debug, Clone, Copy, PartialEq, Eq)]`:
   - Variants: `Wayland`, `X11`, `Xvfb`, `Headless`

4. **Frame struct:**
   - Fields: `pub frame_type: u8`, `pub payload: Vec<u8>`
   - `pub fn new_png(payload: Vec<u8>) -> Self` — convenience constructor setting frame_type to FRAME_TYPE_PNG
   - `pub fn to_bytes(&self) -> Result<Vec<u8>, FrameError>` — returns Err(TooLarge) if payload.len() > u32::MAX as usize. Writes: MAGIC[0], MAGIC[1], VERSION, frame_type, (payload.len() as u32).to_be_bytes(), payload
   - `pub fn from_bytes(data: &[u8]) -> Result<Frame, FrameError>` — validates magic, version, reads frame_type, reads u32 BE length, checks data has enough bytes for payload (return LengthMismatch if not), copies payload slice into Frame

5. **Unit tests** in `#[cfg(test)] mod tests`:
   - `round_trip_png_frame` — create Frame with PNG magic bytes as payload, encode, decode, assert payload identical
   - `round_trip_empty_payload` — frame with 0-byte payload round-trips correctly
   - `round_trip_large_payload` — frame with 64KB payload round-trips correctly
   - `rejects_invalid_magic` — corrupt magic bytes → FrameError::InvalidMagic
   - `rejects_unsupported_version` — corrupt version byte → FrameError::UnsupportedVersion(0xFF)
   - `rejects_truncated_header` — less than 8 bytes → FrameError::TooShort
   - `rejects_truncated_payload` — header says 100 bytes but only 10 present → FrameError::LengthMismatch
   - `display_environment_equality` — verify DisplayEnvironment::X11 == DisplayEnvironment::X11, X11 != Wayland

Run `cargo test 2>&1` and verify all tests pass with zero warnings.
